1. How do you integrate CPT into a DevOps pipeline?

"We 'Shift Left' by embedding automated security scans (SAST/DAST) directly into the pipeline for instant feedback. To preserve deployment speed, we conduct manual CPT assessments asynchronously on staging environments before major releases, ensuring deep security checks without blocking daily velocity."

2. How do you handle secrets management in cloud environments?

"We strictly avoid hardcoding credentials. Instead, we use centralized tools like HashiCorp Vault or AWS Secrets Manager to inject secrets dynamically at runtime, ensuring our code repositories remain clean and secure."

3. Can you ensure zero downtime during security patching?

"Yes. We utilize Blue/Green deployments and Rolling Updates. We deploy patched instances in parallel and only switch traffic once the new instances pass health checks, ensuring continuous service availability."

4. How does Immutable Infrastructure improve security?

"Immutable infrastructure improves security by ensuring that servers or components are never modified after deployment. Any updates or changes require redeploying a new instance, which reduces configuration drift, prevents unauthorized changes, and limits the attack surface."

5. How do you secure containers in a Kubernetes environment?

"We employ a defense-in-depth strategy by using minimal 'distroless' base images to reduce the attack surface and enforcing strict Network Policies to prevent lateral movement. Additionally, we integrate runtime security tools (like Falco) to detect and block anomalous behavior in real-time."
